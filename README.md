# Biblioteca_Bokeh_Juliana_Alzate_20232020082


En este repositorio yo explico la librería **Bokeh**, sus funciones principales y su aplicación para crear visualizaciones interactivas en Python. Analizo sus ventajas frente a otras librerías, como la facilidad para generar gráficos dinámicos y explorables, y comparto mi aprendizaje personal sobre cómo Bokeh facilita comunicar datos de forma clara y efectiva.

Como apoyo práctico, incluyo un programa completo organizado en los módulos **media.py**, **mediana.py**, **moda.py**, **libreria_bokeh.py** y **run.py**, donde aplico Bokeh para representar datos estadísticos. Además, agrego un documento en LaTeX con la explicación teórica y el desarrollo formal del proyecto, junto con un apoyo visual en diapositivas.
